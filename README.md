# Lightweight Linux System Health Checker

A native command-line utility built in Bash to quickly capture core operating system resource utilization metrics. 

## 📊 Features
* **Uptime Log:** Captures current system execution time, logged-in user counts, and CPU load averages.
* **Disk Space Mapping:** Isolates physical storage block utilization and filters out core virtual mounts.
* **Memory Diagnostics:** Pulls real-time, human-readable RAM consumption configurations.

## 🚀 Usage & Quickstart

1. Download or clone this script to your local Linux machine.
2. Grant execution permissions to the script:
   ```bash
   chmod +x systemchecker.sh
